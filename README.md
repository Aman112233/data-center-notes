# Data Center Notes

Personal notes I'm building while working as a Data Center Operations Engineer. I'm documenting what I learn on shift and from study — covering physical infrastructure, power, cooling, networking, and the operational side of running a DC.

The goal is twofold: solidify my own understanding by writing things down properly, and build a public reference I can come back to (and that other early-career DC engineers might find useful).

## Background

Entry-level DC Ops Engineer based in Saudi Arabia. CS/IT/AI background. Working toward infrastructure-adjacent software roles — SRE, DevOps, Platform Engineering, DCIM.
